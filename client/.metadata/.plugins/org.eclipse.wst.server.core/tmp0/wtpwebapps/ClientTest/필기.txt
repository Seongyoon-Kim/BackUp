HTML, Hyper Text Markup Language
- 하이퍼 텍스트 마크업 언어
- Hyper Text: 링크(Link)
- Markup Language: 태그(Tag)를 사용해서 문법을 표기하는 언어
    - SGML, XML, HTML, XHTML 등
- 프로그래밍 언어(X) -> 표현 언어(O): 브라우저 화면에 무언가를 출력하는 기능만 있는 언어
- 난이도 하 -> 개발자 입장 -> 중하
- http://w3.org, http://w3c.org > 웹 관련 기술 표준화 작업
